================================================================================
                          QUICK START (5 MINUTES)
================================================================================

1. Install dependencies:
   npm install

2. Create .env file:
   cp .env.example .env

3. Add Neon database URL to .env:
   DATABASE_URL=postgresql://...

4. Start server:
   npm start

5. Open browser:
   http://localhost:5000

================================================================================
                        DEPLOYMENT (15 MINUTES)
================================================================================

1. Create Neon database (free):
   - Go to neon.tech
   - Create account
   - Create project
   - Copy connection string

2. Deploy to Render (free):
   - Push code to GitHub
   - Connect GitHub to Render
   - Add DATABASE_URL environment variable
   - Deploy!

3. Your live URL:
   https://truth-engine.onrender.com

See DEPLOYMENT_SUMMARY.md for detailed steps.
