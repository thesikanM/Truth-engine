## 💡 Answers to Common Questions

### Q: Why did you choose this technology stack?
**A:** Node.js is beginner-friendly, Express is lightweight, PostgreSQL is reliable, and Neon provides free hosting. Perfect for learning full-stack development.

### Q: How does the trust score algorithm work?
**A:** It evaluates 5 factors (content length, URL credibility, sentiment balance, detail level, language quality) and applies anti-gaming penalties. Each factor contributes 0-15 points.

### Q: Can this be deployed to production?
**A:** Yes! It's production-ready. Can be deployed to Render (backend) and Vercel (frontend) with Neon database.

### Q: What are the limitations?
**A:** No user authentication (demo), basic NLP, English only, no rate limiting. These can be added in future versions.

### Q: How would you improve this?
**A:** Add machine learning, multi-language support, user authentication, advanced analytics, mobile app, and real-time notifications.

### Q: How long did this take?
**A:** ~8-10 hours for development, testing, and documentation.
